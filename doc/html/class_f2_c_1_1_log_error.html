Detailed Description

Writing log files and exception handler.
.


Debug Version (with _DEBUG Macro): 

 #define SETLOGERROR(error,string) error.setError(__FILE__,__FUNCTION__,__LINE__,string)

Sets the error-string, filename, line, and the function name.

Debug Version (without _DEBUG Macro): 

 #define SETLOGERROR(error,string) error.setError(string)

Sets only error-string.
