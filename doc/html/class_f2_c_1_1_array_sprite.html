Detailed Description

Renders a picture(Bitmap), as an array of different values.

Please note the following OpenGL capability are enabled when rendering and at the end disable:
GL_SCISSOR_TEST (if necessary)
GL_STENCIL_TEST
GL_TEXTURE_2D
GL_DEPTH_TEST
GL_ALPHA_TEST
GL_BLEND
and use glShadeModel(GL_SMOOTH)

Dont forget to set the depth and alpha test function, e.g.:
glAlphaFunc(GL_GREATER, 0.0f)
glDepthFunc(GL_LEQUAL)

Used OpenGL Buffers
-GL_COLOR_BUFFER_BIT
-GL_DEPTH_BUFFER_BIT
-GL_STENCIL_BUFFER_BIT
