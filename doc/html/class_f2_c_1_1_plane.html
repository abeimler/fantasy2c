Detailed Description

Renders an image (Bitmap class) on the screen as an infinite Scrollable image.

Please note the following OpenGL capability are enabled when rendering and at the end disable:
GL_SCISSOR_TEST (if necessary)
GL_STENCIL_TEST
GL_TEXTURE_2D
GL_DEPTH_TEST
GL_ALPHA_TEST
GL_BLEND
and use glShadeModel(GL_SMOOTH)

Dont forget to set the depth and alpha test function, e.g.:
glAlphaFunc(GL_GREATER, 0.0f)
glDepthFunc(GL_LEQUAL)

Used OpenGL Buffers
-GL_COLOR_BUFFER_BIT
-GL_DEPTH_BUFFER_BIT
-GL_STENCIL_BUFFER_BIT

Render the bitmap on screen.

Returns:
-1, show is false (or from viewport).
-2, The width or height of the bitmap or the src_rect are 0.
-3, The sprite is outside the display (glViewport).
-4, The alpha color of all 4 Vertices are less than 1.
-5, zoom_x or zoom_y is exactly 0.
1, Sprite has been successfully rendered.
